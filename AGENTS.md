## Project Overview

This is an **Astro** website with TypeScript, deployed to Cloudflare Workers.

## Success Criteria

**All checks must pass before completing work:**

```bash
# 1. Format check
npm run format:check

# 2. Lint check
npm run lint

# 3. Type check
npm run typecheck

# 4. Build
npm run build
```

## Key Technologies

- **Astro**: Static site generation
- **TypeScript**: Type-safe JavaScript
- **Cloudflare**: Edge deployment
- **Content Collections**: Type-safe content with `src/content.config.ts`

## Code Style

- **No Comments**: Do not add code comments. Code should be self-explanatory.
